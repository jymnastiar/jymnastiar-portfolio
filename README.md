# 🌌 Jymnastiar - Personal Portfolio

Welcome to the repository of my personal portfolio website. This is a premium, interactive, and immersive 3D single-page portfolio designed to showcase my skills, projects, professional experience, and academic achievements.

Built with modern web technologies, it features an interactive 3D robot scene, ultra-smooth scrolling, and physics-based/scroll-triggered animations to deliver a high-quality user experience.

## ✨ Features

- **🤖 3D Interactive Scene:** Integrated with a custom 3D Robot scene powered by **Spline** for an engaging hero section.
- **🌀 Scroll-Triggered Animations:** Powered by **GSAP (GreenSock)** and **ScrollTrigger** for smooth entrance, exit, and hover animations.
- **📜 Smooth Scrolling:** Implemented using **Lenis** to provide modern momentum-based scrolling across all modern browsers.
- **🎨 Modern Aesthetics:** Built using **Tailwind CSS v4** with a sleek dark theme, glassmorphism, responsive grid layouts, and custom interactive states.
- **📱 Responsive Layout:** Fully optimized for seamless browsing across desktop, tablet, and mobile devices.
- **⚡ High Performance:** Built on **Vite + React 19** and TypeScript, linted with **Oxlint** for ultra-fast load times and developer experience.

## 🛠️ Tech Stack & Libraries

### Frontend Core

- **Framework:** React 19 (TypeScript)
- **Bundler & Dev Server:** Vite
- **Styling:** Tailwind CSS v4
- **Routing:** React Router DOM (v7)

### Animation & 3D

- **3D Renderer:** `@splinetool/react-spline` & `Three.js` / `@react-three/fiber`
- **Animation Engine:** GSAP (GreenSock) & `@gsap/react`
- **Scroll Interactions:** GSAP ScrollTrigger
- **Smooth Scroll:** Lenis

### Linting & Tooling

- **Linter:** Oxlint (Fastest JS/TS linter)
- **Package Manager:** pnpm

## 🛠️ Getting Started & Local Development

This project uses `pnpm` as the package manager. Follow the steps below to set up the project locally:

### 1. Prerequisites

Make sure you have **Node.js** (v18+ recommended) and **pnpm** installed on your system.

```bash
# To install pnpm globally (if not already installed)
npm install -g pnpm
```

### 2. Clone the Repository

```bash
git clone https://github.com/jymnastiar/jymnastiar-portfolio.git
cd jymnastiar-portfolio
```

### 3. Install Dependencies

```bash
pnpm install
```

### 4. Run Development Server

```bash
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the application.

### 5. Build for Production

```bash
pnpm build
```
